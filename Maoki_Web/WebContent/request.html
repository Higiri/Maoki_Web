動画制作の基本料金の目安は、次の通りです。

内容料金
エンコードのみ無償
歌詞入れのみ3000円～
ワンコーラス
Twitter動画

オリジナル

次の内容の場合は、追加料金が必要です。

内容追加料金
コラボ・合唱動画500円×人数
一週間以内のスピード納品5000円
商業利用・収益目的要相談
注意

この料金は、あくまで目安です。適宜ご相談ください。

スピード納品の場合、あまり高度な動画は制作できません。ご了承ください。

お支払い方法

料金のお支払い方法は、Amazonギフト券です。

お支払い後の返金はいかなる理由でも致しかねます。

依頼の流れ

動画制作の大まかな流れは、次の通りです。

  • TwitterへのDM、またはメールに依頼内容の送付
  • 納期・料金の確認
  • 制作（通常一ヶ月～）
  • 納品・お支払い
ご依頼にあたっての注意事項
  • ご依頼はTwitter、メールでのみ受け付けます。
    (進捗などの連絡をDiscord等で行うことは可能です)
  • 動画のリテイクは何度でも無償です。
    (納期直前のリテイクは、納期に間に合わなくなる場合があります。)
  • 納品前であれば、キャンセルも可能です。
  • 特に指定がない場合、納品する動画は次の形式となります。
形式MPEG-4 H.264（.mp4）
総ビットレート15000kbps
音声480kbps
制作に必要なファイルについて
  • 音源・立ち絵素材などは、zip形式にまとめ、アップローダー等を経由してお渡しください。
  • 歌詞などがある場合は、そちらも同梱してださい
  • 仮の絵しかない場合でも、先行して制作が可能な場合があります。ご相談ください。
