依頼について
  • 依頼
    依頼の方法

    TwitterのDM(ダイレクトメッセージ)、またはメールで依頼内容を送信してください。

  • 料金
    料金について

    基本料金の目安は次の通りです。

    内容基本料金
    エンコードのみ無償
    歌詞入れのみ3000円～
    ワンコーラス
    Twitter動画
    フルMV5000円～
    オリジナル

    次の内容の場合、基本料金に加えて追加料金が必要となります。

    内容追加料金
    コラボ・合唱動画500円×人数
    一週間以内のスピード納品5000円
    商業利用・収益目的要相談
    注意

    この料金は目安です。

    スピード納品の場合、あまり高度な動画は制作できません。

    表にない内容でも制作可能な場合がありますので、お気軽にご連絡ください。

  • 制作
    制作に必要な期間

    フルMV、オリジナル動画の制作には、通常、一ヶ月以上の期間を必要とします。

    必要な素材

    音源や立ち絵イラスト、歌詞などの素材は、まとめてzip形式で圧縮し、アップローダーやクラウド等を経由してお渡しください。

    仮の絵しかなくても、先行して制作を始められる場合があります。ご相談ください。

    リテイク（修正等）

    動画のリテイクは基本的に無償で行っています。
    (納期直前のリテイクは、納期に間に合わなくなる場合があります。)

    キャンセル

    納品前であれば、キャンセルも可能です。

  • 納品とお支払い
    納品

    形式の指定がない場合、納品する動画は次の形式となります。

    形式MPEG-4 H.264（.mp4）
    画素1080p
    総ビットレート15000kbps
    音声480kbps
    お支払い

    現在対応しているお支払い方法は、Amazonギフト券です。

    お支払い後の返金はいかなる理由でも致しかねます。
